```shell
Command mode:
dd	=> xóa 1 dòng 
<n>dd	=> xóa n dòng
yy	=> copy 1 dòng
<n>yy	=> copy n dòng
p	=> paste
u	=> undo
...
Last line mode:
:q!	=> quit (không lưu)
:w	=> write (save)
:wq!	=> write & quit
:x	=> write & quit
:w newfile	=> save as ...
:set nu	=> Hiển thị số thứ tự dòng 
:set nonu
Hoặc
:set number
:set nonumber
``````
