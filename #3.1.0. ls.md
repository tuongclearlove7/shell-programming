```shell
Xem nội dung thư mục:
	ls [options] [path]
Với các options:
-l	//Xem ở dạng đầy đủ
-d	//Xem chính folder được chỉ định
-a	//Xem file ẩn
-i	//Xem inode number

//Ngoài ra thì có thể dùng các công cụ khác như: dir, tree

VD: 
	ls -la /dev
	ls -lai /dev
	ls -i
	ls -ld /home
Lưu ý: Có 1 lệnh là alias (shortcut của lệnh) của lệnh ls -l; đó là lệnh ll
``````
